# Wordchipper Book

Requires [mdbook](https://rust-lang.github.io/mdBook/guide/installation.html):

```bash
cargo install mdbook
```

## Serve locally (with live reload)

```bash
mdbook serve
```

Opens at [http://localhost:3000](http://localhost:3000).

## Build static HTML

```bash
mdbook build
```

Output is in `book/book/`.
